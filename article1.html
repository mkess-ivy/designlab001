Splitting the Frame In a Meaningful Way
Written by SunnyOwl
Cinematographers and directors love adding "lines" to the frame. Doorways, windows, corners, pillars all of them are used to create rich and interesting compositions. You can hardly find an action scene in Michael Bay's movie, which doesn't include lampposts. In the extreme case, all these straight lines can effectively split the frame into several parts. And, if executed brilliantly, this split will be meaningful and convey extra information. Sometimes the frame composition emphasizes what is happening in the scene using a visual language.
